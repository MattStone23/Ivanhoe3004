"WITHDRAW"
"ENDTURN"
"STARTTOURN|colour" 
"WINTOKEN|colour" (for winning purple tournament only)
"LOSETOKEN|colour" (for withdrawing with maiden only)
"CHAT|message"
"DESC|card"

cards are the format "colourNumber"
for example A15. P4.

colours are just a single character. P,B,R,G,Y,W

SPECIAL SERVER COMMANDS
"QUIT" this will close the client connection
"SHUTDOWN" this will shutdown the server if sent from the host
